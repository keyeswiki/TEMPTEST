软件下载
========

1. 打开\ `Software \|
   Arduino <https://www.arduino.cc/en/software>`__\ 下载软件，然后选择对应的系统下载，下面以window系统为例。（\ |image-20250527113703588|)

   .. figure:: ./media/7ea915dd-dfa2-4b18-ae8a-853082fd85ad.png
      :alt: 7ea915dd-dfa2-4b18-ae8a-853082fd85ad

      7ea915dd-dfa2-4b18-ae8a-853082fd85ad

2. 然后选择\ |image-20250527113750160|\ ，再一次选择\ |image-20250527113816499|\ ，就可以看到正在下载的页面\ |image-20250526175651303|\ 。

   .. figure:: ./media/image-20250527115114993.png
      :alt: image-20250527115114993

      image-20250527115114993

   .. rubric:: 软件安装
      :name: 软件安装

   1. 点击此处文件夹\ |image-20250527115344256|\ 进入到下载中心，双击\ |image-20250527115501137|\ 进行安装。

   2. 选择\ |image-20250527115748575|\ ，跳转页面后选择\ |image-20250527120154571|,再点击下一步\ |image-20250527120226617|\ 。

      .. figure:: ./media/image-20250527121512549.png
         :alt: image-20250527121512549

         image-20250527121512549

   3. 跳转页面后，点击\ |image-20250527122109892|\ ，可把软件放到指定位置，点击安装\ |image-20250527122524648|\ ，安装完成后，点击\ |image-20250527122855480|\ 。

      .. figure:: ./media/image-20250527122939211.png
         :alt: image-20250527122939211

         image-20250527122939211
