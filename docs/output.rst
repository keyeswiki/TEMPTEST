说明
====

这款套件包含我们玩单片机时常用到的24款传感器模块，如有源蜂鸣器模块、5V继电器模块、温湿度传感器等等。它适用于各种单片机和树莓派。同时我们为套件里的每一款传感器都编写了资料，资料里介绍了传感器模块基于Arduino开发板的使用方法，包含了接线方法、测试代码、实验结果等信息。
